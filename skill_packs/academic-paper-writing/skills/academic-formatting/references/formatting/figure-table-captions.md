# Figure and Table Caption Checks

Use this guide when a draft contains figures, images, tables, screenshots, charts, or experimental result tables.

## Figure Rules

- Each figure should have a numbered caption, for example `图 1 系统总体架构`.
- If Markdown image syntax appears without a nearby numbered caption, record `figure_caption_missing`.
- Do not invent a figure title from thin air. If the caption is unclear, preserve the image placeholder and add a warning.

## Table Rules

- Each table should have a numbered caption, for example `表 1 实验配置对比`.
- If Markdown table syntax appears without a numbered caption, record `table_caption_missing`.
- Do not fabricate table data or statistical results.

## Cross References

Skill5 may normalize label spelling and spacing, such as `图1` to `图 1`, but it must not change the interpretation of the figure/table.
