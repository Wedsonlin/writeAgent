# Plagiarism Optimization Boundary

Skill6 can help reduce repetitive and template-like expression, but it does not run a commercial plagiarism system unless the user provides a real plagiarism report.

## Allowed

- Identify repeated phrases across sections.
- Suggest alternative academic wording.
- Rewrite common template sentences into paper-specific expressions.
- Preserve citations while varying sentence structure.
- Record suggestions in `plagiarism_optimization[]`.

## Not Allowed

- Claim a numeric similarity score without an actual report.
- Claim the paper has passed plagiarism detection.
- Remove citations to make text appear more original.
- Change technical claims, data, or source attributions to avoid similarity.
- Replace precise terminology with vague wording that damages academic meaning.

When no external report is supplied, describe the result as “查重优化建议” or “重复表达改写建议”, not as a verified plagiarism check.
