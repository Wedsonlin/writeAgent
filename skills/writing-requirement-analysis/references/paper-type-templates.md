# 论文类型 → 章节模板速查

`chapter_template.py` 实现了下表的硬编码版本；新增论文类型时请同步更新两侧。

| paper_type   | 章节数 | 章节顺序                                                                 | 字数权重（示例） |
| ------------ | ------ | ------------------------------------------------------------------------ | ---------------- |
| `survey`     | 7      | 引言 / 概念 / 代表方法 / 对比 / 应用 / 挑战与方向 / 结论                 | 8/10/30/20/12/12/8 |
| `empirical`  | 6      | 引言 / 相关工作 / 方法 / 结果 / 讨论 / 结论                              | 10/15/25/30/13/7  |
| `theoretical`| 7      | 引言 / 预备知识 / 理论模型 / 性质扩展 / 示例 / 讨论 / 结论                | 10/10/30/20/12/10/8 |
| `system`     | 8      | 引言 / 相关工作 / 系统设计 / 实现 / 评测 / 案例 / 讨论 / 结论            | 8/10/20/22/15/12/8/5 |
| `case_study` | 6      | 引言 / 相关工作 / 案例 / 分析 / 讨论 / 结论                              | 10/15/25/25/15/10  |
| `position`   | 5      | 立场 / 现状 / 论证 / 反驳与回应 / 结论                                    | 15/20/30/20/15     |

## 选型建议

- 用户说"综述/综览/Review/Survey" → `survey`。
- 用户说"实验/实证/对照研究/effect of ..." → `empirical`。
- 用户说"新算法/新模型/收敛性证明/理论分析" → `theoretical`。
- 用户说"系统/平台/工具实现/Demo" → `system`。
- 用户说"案例/教学案例/Case Study" → `case_study`。
- 用户说"立场/呼吁/观点/Position" → `position`。

不确定时默认 `survey`（覆盖面最广）。
